Combinatorics

Combinatorial mathematics is the field of mathematics concerned with problems of selection, arrangement, and operation within a finite or discrete system. Included is the closely related area of combinatorial geometry. One of the basic problems of combinatorics is to determine the number of possible configurations (e.g., graphs, designs, arrays) of a given type. Even when the rules specifying the configuration are relatively simple, enumeration may sometimes present formidable difficulties. The code below shows the way to find the number of ways to place K bishops on an N×N chessboard so that no two bishops attack each other. For more details, click here. 

            int squares (int i) {
                if (i & 1)
                    return i / 4 * 2 + 1;
                else
                    return (i - 1) / 4 * 2 + 2;
            }
            int bishop_placements(int N, int K){
                if (K > 2 * N - 1)
                    return 0;

                vector> D(N * 2, vector(K + 1));
                for (int i = 0; i < N * 2; ++i)
                    D[i][0] = 1;
                D[1][1] = 1;
                for (int i = 2; i < N * 2; ++i)
                    for (int j = 1; j <= K; ++j)
                        D[i][j] = D[i-2][j] + D[i-2][j-1] * (squares(i) - j + 1);

                int ans = 0;
                for (int i = 0; i <= K; ++i)
                    ans += D[N*2-1][i] * D[N*2-2][K-i];
                return ans;
            }
